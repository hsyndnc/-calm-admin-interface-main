# Calm Admin Dashboard

A modern, responsive admin dashboard built with React, TypeScript, and Vite.

## Tech Stack

- **Framework**: [React](https://react.dev/) (Vite powered)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: [TanStack Query](https://tanstack.com/query)
- **Routing**: [React Router DOM](https://reactrouter.com/)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for Production

```sh
# Build the project
npm run build

# Preview the production build
npm run preview
```

## Project Structure

- `src/components/`: Reusable UI components.
- `src/pages/`: Application views and pages.
- `src/hooks/`: Custom React hooks.
- `src/lib/`: Helper utilities and configurations.

## Testing

```sh
# Run unit tests
npm test

# Run End-to-End tests
npx playwright test
```
